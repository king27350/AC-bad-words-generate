## Getting Started
### Installing

下載專案
```
$git clone https://github.com/king27350/AC-bad-words-generate.git
```
安裝套件
```
npm install
```
### Running the test
開啟專案
```
npm run dev
```
開啟網頁輸入
```
http://localhost:3000
```
